# Proyecto: Arreglos Multidimensionales en Ingeniería

## 📌 Descripción
Este proyecto resuelve problemas reales de ingeniería utilizando arreglos multidimensionales en Python. Se aplican conceptos en estructuras, fluidos, imágenes médicas, sensores y transformaciones geométricas.

---

## 🧩 Problemas resueltos

### 🔹 Problema 1: Análisis de fuerzas
Se usa una matriz 3x3 para representar fuerzas en nodos de una estructura.
Se calcula la reacción necesaria para mantener el equilibrio.

### 🔹 Problema 2: Simulación de fluido
Se usa un arreglo tridimensional (3x3x3) para representar la presión de un fluido.
Se actualiza cada celda usando el promedio de sus vecinos.

### 🔹 Problema 3: Imágenes médicas 3D
Se aplica un filtro de suavizado a un volumen tridimensional.
Esto reduce el ruido en los datos.

### 🔹 Problema 4: Sensores
Se analizan datos de temperatura en una matriz 5x5.
Se calculan promedios y desviaciones estándar por filas y columnas.

### 🔹 Problema 5: Transformaciones
Se representan puntos en un plano cartesiano.
Se aplica una rotación usando matrices y se visualizan los resultados.

---

## ⚙️ Enfoque y eficiencia
El uso de arreglos multidimensionales permite:
- Acceso rápido a los datos
- Operaciones vectorizadas con NumPy
- Mayor eficiencia en cálculos científicos
- Código más organizado y escalable
