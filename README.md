# Proyecto: Arreglos Multidimensionales en Ingeniería

## Descripción
Este proyecto resuelve problemas reales de ingeniería utilizando arreglos multidimensionales en Python.

## Problemas resueltos

### 1. Análisis de fuerzas
Se usa una matriz 3x3 para representar fuerzas en nodos.
Se calcula la reacción necesaria para equilibrio.

### 2. Simulación de fluido
Se usa un arreglo 3D para simular presión en un fluido.
Se actualiza usando promedio de vecinos.

### 3. Imágenes médicas
Se aplica un filtro de suavizado en un volumen 3D.
Reduce el ruido en los datos.

### 4. Sensores
Se analizan datos 5x5 de temperatura.
Se calculan promedios y desviaciones estándar.

### 5. Transformaciones
Se aplican transformaciones lineales a puntos 2D.
Se visualiza la rotación.

## Enfoque y eficiencia
El uso de arreglos multidimensionales permite:
- Acceso rápido a datos
- Operaciones vectorizadas (NumPy)
- Mejor rendimiento en cálculos científicos
